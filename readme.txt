== Description ==
Wemake Chat helps users to add WhatsApp chat button.
Simple to use, no additional functions required.

== Installation ==

1. Upload `wemake-chat` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to plugin settings page in the admin menu and set Whatsapp number
